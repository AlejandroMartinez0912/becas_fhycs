# Becas FHyCS — Sistema de gestión de becas universitarias

Sistema desarrollado para la Facultad de Humanidades y Ciencias Sociales (FHyCS) de la Universidad Nacional de Misiones (UNaM).

> **Nota:** El código fuente es propiedad de la institución y no se encuentra disponible públicamente.

---

## Descripción

Plataforma web que digitaliza y centraliza el proceso completo de gestión de becas universitarias: desde la recepción de solicitudes hasta la asignación final, reemplazando los procesos manuales en papel.

---

## Funcionalidades principales

- Registro y seguimiento de solicitudes de becas por parte de los estudiantes
- Panel de evaluación para el equipo de bienestar estudiantil
- Flujo de aprobación con múltiples estados (pendiente → en evaluación → aprobada / rechazada)
- Asignación y renovación de becas activas
- Generación de reportes por período, tipo de beca y estado
